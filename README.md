# Cybersecurity Portfolio

Repository ini berisi portofolio cybersecurity yang mendokumentasikan proses security assessment secara simulasi di lingkungan lab yang terkontrol dan terizin.

Portofolio ini dibuat untuk menunjukkan pemahaman alur kerja cybersecurity, mulai dari pengenalan target, scanning, dokumentasi eksploitasi (dengan bukti yang disanitasi), analisis post-exploitation secara konseptual, sampai rekomendasi mitigasi keamanan.

Sebagian dokumentasi teknis eksploitasi direferensikan ke artikel eksternal berbasis lab untuk menghindari duplikasi dan menjaga batas etika.

## Scope Pengujian
- Lingkungan lab (virtual machine)
- Sistem yang memang dibuat rentan
- Tidak ada pengujian ke sistem publik atau produksi

## Ethical Disclaimer
Semua aktivitas yang didokumentasikan di repository ini dilakukan secara etis, legal, dan hanya untuk keperluan akademik.  
Tidak ada sistem nyata, data asli, atau jaringan publik yang diserang.

## Struktur Portofolio
- Pengantar dan scope
- Reconnaissance & scanning
- Dokumentasi eksploitasi (referensi eksternal)
- Post-exploitation & privilege escalation (konseptual)
- Laporan akhir dan mitigasi
